# Changelog

All notable changes to this project are documented in this file.

## [v0.1.4] - 2026-03-10

### Added
- Custom Precision Anvil menu registration and opening via right click on anvils.
- Compact client-side anvil cost display (`used / required`) with color feedback.
- Configurable server-side cost model for required and consumed levels.

### Changed
- Anvil result pickup checks now rely on configurable required levels.
- XP consumption on result take now applies configurable used levels separately.

### Notes
- This is the current latest release baseline for the repository migration to GitHub.
